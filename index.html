Position

IDEAS PRINCIPALES

Propiedad Position

Especifica el tipo de método de posicionamiento utilizado para un elemento , hay 5 valores de posición diferentes:
Static, relative, fixed, absolut y sticky.
Los elementos se posicionan usando las propiedades de top, bottom, left y right.

Un elemento con position:static; no esta posicionado de ninguna manera especial, siempre se posiciona de acuerdo con el flujo normal de la pagina.

El posicionamiento relativo desplaza un elemento respecto de su posición original establecida mediante el posicionamiento normal.
Cuando un elemento se posiciona de forma absoluta, el resto de elementos de la página se ven afectados y modifican su posición. Los demás elementos ocuparan el espacio que deja el elemento desplazado y es probable que se produzcan soplamientos con otros elementos.

Position:fixed; Un elemento de position:fixed; esta ubicado en relación con la ventana grafica, lo que significa que siempre permanece en el mismo lugar, incluso si la pagina se desplaza.
Las propiedades: top, bottom, left y right, se utilizan para posicionar el elemento
Position:sticky;
Un elemento con position:sticky;se posiciona en función de la posición de desplazamiento del usuario.
Un elelem nto sticky alterna entre relativo y fijo, dependiendo de la posición de desplazamiento. Se posiciona de forma relativa hasta que se encuentra una posición de desplazamiento deda en ela ventana grafica, luego se “pega” en su lugar (como position:fixed;)

Notas de clase

Position:static;
Los elementos HTML se posicionan estáticos de forma predeterminada . Los elementos posicionados estáticos no se ven afectados por las propiedades top, bottom left, right y z-index.
Position:relative;
Un elemento con position:relative; esta posicionado en relación a su posición normal.
El desplazamiento de la caja se controla con las propiedades top, bottom, left,right y z-index.Los demás elementos no ocuparan el espacio que deja el elemento desplazado.
Position:absolute;
El posicionamiento absoluto se emplea para establecer de forma exacta la posición en la que se muestra la caja de un elemento.
La nueva posición del elemento se indica mediante las propiedades top, bottom, left, rigth y z-indexque en este caso dependen del posicionamiento del elemento contenedor cuando tiene una posición diferente a static.

RESUMEN

Estas propiedades las podemos usar para agregar un boton, una imagen, un video, etc y que se puedan quedar en una posicion fija, que se puedan desplazar de una forma normal o que se se pueda quedar en la forma exacta de la que muestra la caja.

Display

IDEAS PRINCIPALES

Propiedad display

Establece el tipo de caja que genera cada elemento, traen un valor de display que ya está predeterminado.


Displey:none;
Se usa comúnmente conJavaScript para ocultar y mostrar elemento cuando el usuario realiza alguna acción como pulsar un botón o un enlace sin eliminarlos mi crearlos.

Display:inline-block; permite establecer un ancho y una altura en el elementó. Se respetan los margin y padding que con display: inline no son respetados, NO agrega salto de líneas después del elemento.Puede alinearse junto a otros elementos.
Displey:none; Hace que el elemento no genere ninguna caja, el resultado es que el elemento desaparece por completo de la página y no ocupa sitio, por lo que los elementos adyacentes ocupan su lugar. Se utiliza sobre un elemento todos sus descendientes también desaparecen por completo de la pagina.

Notas de clase

Display:block; siempre comienza en una nueva línea y ocupa todo el ancho disponible. Elementos con display block DIV H1 P FROM HEADER

Display:inline; no comienza en una nueva línea y solo ocupa el ancho que sea necesario. no es necesario poner el width y heigth por que usa solo necesario, ya que esta predeterminando Elementos en línea SPAN A IMG

Display:table; permite asignar una comportamiento de tabla a los elementos que haya dentro, se pueden usar otra seria de valores de display que corresponden con los roles de distintas estiquetas HTML que usas en las tablas tradicionales
Display :table-row; Actúa como el elemento fila -tr-
Display:table-cell; etiquetas que trabajan como una celda -td-

RESUMEN

Con la propiedad display, podemos crear tablas, podemos crear cajas con medidas predeterminadas

Animacion

IDEAS PRINCIPALES

Css permite animar la mayoría de los elementos html sin usar JavaScript
• Cuando especifica estilos css dentro de la regla @keyframes, la animación cambiara del estitilo actual al estilo nuevo
Sintaxis
@keyframes nombre { From{background-color:red; Width:100px; } To {background-color:yellow; Width:500px; } }
Sintaxis 2
@keyframes nombre{ 0%{backgroun-color:red; Width:100px; } 50%{background-color:yellow; Width:500px; } 100%{backgroun-color:yellow; Width:500px; } }
Para que una animación funcione, se debe vincular la animación a un elemento.
Div{ Animation-name:animo1; }
@keyframes animo1{ From{background-color:red;} To {background-color:yellow;} }

• Animation : propiedad rápida para configurar todas las propiedades de la animación. Por ejemplo, los siguientes fragmentos expresan lo mismo: Div{ Animation:ejemplo 5s linear 2s infinitive alterne; } Div{ Animation-name:ejemplo; Animation-duration:5s; Animation-timining-function;linear; Animation-delay:2s; Animation-interation.count:infinite; Animation-direction:alternate; }

    • Animationduration
    Especifica cuanto tiempo debe transcurrir una animación para completar el ciclo. Si no se especifica la propiedad de duración de la animación, no se producirá ninguna animación, ya que el valor predeterminado es 0 segundos. También, se puede especificar usando, cambiara usando”to (0%inicio)” y “from(100%, completo)”.
    • Animation-interation-count: Especifica el numero de veces que se debe reproducir una animación. Se expresa con números cerrados y se puede usar también infinito.

Notas de clase

    • Animation-delay Especifica un retraso para el inicio de una animación. Se expresa en seundos. Se puede utilizar unidades negativas
    • Animation-play-state: Especifica si la animación se esta ejecutando(running) o esta en pausa (paused)
    • Animation- timing-function: Especifica la curva de la velocidad de la animación - Linear, Ease, Ease-in, Ease-out,Ease-in-out, Cubic-bezier
    • Animation-direction: Especifica si una animación debe reproducirse hacia adelante, hacia atrás, o en ciclos alternos. - Normal, Reverse, Alternate, Alternate-reverse
    • Animation-fill-mode: Especifica un estilo para el elemento cuando la animación no se esta reproduciendo (antes de que comience, después de que termine o ambos) - None, Forwards,Backwards, Both

    RESUMEN

    Podemos crear animaciones, ya sean normales, que vayan de reversa, que tenga duracion, que se queden pausadas y hacerlas que vayan a una direccion en especifico y de algun color.

    Referencias avanzadas

    IDEAS PRINCIPALES Y NOTAS

    • Selector de atributo Selecciona elementos HTML en función de la presencia o el valor de uno de sus atributos. Esto permite aplicar estilos a los elementos que tienen un atributo especifico, o cuyo atributo tiene un valor determinado
    • Selector de atributo-[atributo^=”inicio”] Una regla con el selector (^=) selecciona elementos cuyo atributo comienza con un valor especifico
    • Selector de atributo- [atributo$=”final”] Una regla con el selector($=)selecciona elementos cuyo atributo termina con un valor especifico
    • Selector de atributo-[atributo*=”cadena”] Una regla con el selector (*=) selecciona elementos cuyo atributo contiene una cadena especifica en cualquier posición

    • Selectores de relación Los selectores de relación en css permiten seleccionar elementos basados en su relación con otros elementos basados en su relación con otros elementos dentro del documento. Estos selectores son útiles para aplicar estilos a elementos que tienen un vinculo estructural con otros elementos, como ancestros, padres, hermanos o descendientes.
    - Selector de hijos directos (>) Selecciona los elementos que son hijos inmediatos de un elemento padre.
    - Selector de hermanos adyacentes(+) Selecciona un elemento que es hermano inmediato, es decir, estájusto después de otro elemento
    - Selector de hermanos generales (~) - Selecciona todos los elementos que son hermano de un elemento anterior, es decir, esta antes, sin importar si son hermanos inmediatos o no

    Notas de clase

    • Atributo id : Es como el un nombre único que identifica al elemento . el valor de este atributo no puede ser duplicado. Este nombre debe ser único en todo el documento.
    • Atributo class Este es mas flexible y puede ser asignadas a varios elementos html en el documento que comparten un diseño similar
    • Elementos Un selector de elementos en css es una regla que selecciona todos los elementos de un tipo especifico en el documento html
    • Pseudoclases-;nth-child() Nos permite encontrar a un hijo especifico, para poder poner algún color o tamaño de letra se puede poner solo agregándole el numero en css
    • Pseudoclases-;nth-child() Nos permite encontrar a un hijo especifico, para poder poner algún color o tamaño de letra se puede poner solo agregándole el numero en css
    • Pseudoclases-:nth-child(odd):nth-child(even) Las palabras clave odd(impar) y even(par) se utilizan para seleccionar elementos con base a posición en la secuencia de hijos, específicamente para alternar entre elementos impares y pares.
    • Pseudoclases-:frist-child Existen otras importantes pseundoclases relacionadas con esta ultima. La pseudoclasefrist-child referencia solo el primer hijo.
    • Pseudoclases-:last-child Hace referencia a solo el ultimo hijo, esta pseudoclase en particular no requieren palabras clave o parámetros
    • Pseudoclases-:only-child Afecta un elemento siempre y cuando sea el único hijo disponible. Estas pseudoclases en particular no requieren palabras clave o parámetros
    • Pseudoclases-:not(#) Nos permite declarar una excepción. Los estilos en la regla cread con esta pseudoclase serán asignados a todo elemento excepto aquellos incluidos en la referencia entre paréntesis

    RESUMEN

    Con estas propiedades podemos ponerle un estilo unico a los parrafos o a los renglones, acomodar los numeros por nuemero pares o impares, o dejar un elemento sin estilo
